## 확장 프로그램이란?

이 확장 프로그램은 다음과 같은 기능을 제공합니다:

- ✅ **자동 추적**: 좋아요, 저장, 북마크, 리트윗을 자동으로 감지하여 저장
- ✅ **다중 플랫폼**: Instagram, Twitter/X, LinkedIn, TikTok 지원
- ✅ **시청 시간 추적**: 각 게시물을 얼마나 오래 봤는지 추적
- ✅ **AI 분류**: OpenAI 또는 Google Gemini를 사용하여 콘텐츠를 자동으로 카테고리별로 분류
- ✅ **로컬 저장**: 모든 데이터는 브라우저에 안전하게 저장 (외부 서버로 전송되지 않음)
- ✅ **대시보드**: 저장된 모든 콘텐츠를 한눈에 보고 관리

## 설치 방법

### 1단계: Chrome에 확장 프로그램 로드하기

1. **Chrome 확장 프로그램 페이지 열기**
   - 주소창에 `chrome://extensions/` 입력
   - 또는 Chrome 메뉴(점 3개) → 더보기 도구 → 확장 프로그램

2. **개발자 모드 활성화**
   - 페이지 오른쪽 상단의 "개발자 모드" 토글을 켜기

3. **확장 프로그램 로드**
   - "압축해제된 확장 프로그램을 로드합니다" 버튼 클릭
   - 다음 폴더 선택: `/Users/zone/Chrome_extension`
   - 확장 프로그램이 목록에 나타나면 성공!

4. **확장 프로그램 고정 (선택사항)**
   - Chrome 툴바의 퍼즐 아이콘 클릭
   - "Social Media Content Tracker" 찾기
   - 핀 아이콘 클릭하여 항상 보이게 설정

## 초기 설정

### 2단계: API 키 받기

AI 분류 기능을 사용하려면 API 키가 필요합니다. **OpenAI와 Gemini 둘 다 입력할 수 있으며, 둘 다 입력하면 OpenAI를 먼저 사용하고 실패 시 Gemini를 자동으로 사용합니다.**

#### 옵션 A: OpenAI API 키 받기

1. https://platform.openai.com/api-keys 방문
2. 계정 생성 또는 로그인
3. "+ Create new secret key" 클릭
4. 키 이름 입력 (예: "Social Media Tracker")
5. 생성된 키 복사 (형식: `sk-proj-...`)

**비용**: 게시물당 약 $0.0001-0.001 (매우 저렴! 약 10,000개 게시물당 $1)

#### 옵션 B: Google Gemini API 키 받기

1. https://makersuite.google.com/app/apikey 방문
2. Google 계정으로 로그인
3. "Create API key" 클릭
4. Google Cloud 프로젝트 선택 또는 생성
5. 생성된 키 복사

**비용**: 무료 티어 (분당 60개 요청), 이후 사용량 기반 과금

#### 옵션 C: .env 파일 사용 (고급 사용자)

1. 프로젝트 루트에 `.env` 파일 생성
2. `.env.example` 파일을 참고하여 다음 형식으로 입력:
   ```
   OPENAI_API_KEY=sk-proj-your-key-here
   GEMINI_API_KEY=your-gemini-key-here
   ```
3. 빌드 스크립트 실행: `node scripts/build-env.js`
4. 또는 `config/env.js` 파일에 직접 값을 입력할 수도 있습니다

### 3단계: 확장 프로그램 설정

#### 방법 1: UI에서 설정 (권장)

1. **확장 프로그램 아이콘 클릭**
   - Chrome 툴바의 보라색 북마크 아이콘 클릭
   - 보이지 않으면: 퍼즐 아이콘 → "Social Media Content Tracker" 고정

2. **설정 열기**
   - 팝업에서 "Settings" 버튼 클릭

3. **설정 입력**
   - **OpenAI API Key**: OpenAI 키 입력 (선택사항)
   - **Google Gemini API Key**: Gemini 키 입력 (선택사항)
   - 💡 **팁**: 두 키를 모두 입력하면 OpenAI를 먼저 사용하고, 실패 시 Gemini를 자동으로 사용합니다
   - **Time Threshold**: 5초로 유지 (기본값 권장)
   - **Enabled Platforms**: 모든 플랫폼 체크 유지

4. **설정 저장**
   - "Save Settings" 버튼 클릭
   - 설정 모달 닫기

#### 방법 2: .env 파일 사용

1. `config/env.js` 파일을 열어서 직접 API 키 입력:
   ```javascript
   const ENV_CONFIG = {
     OPENAI_API_KEY: 'sk-proj-your-key-here',
     GEMINI_API_KEY: 'your-gemini-key-here'
   };
   ```

2. 또는 `.env` 파일 생성 후 빌드 스크립트 실행:
   ```bash
   node scripts/build-env.js
   ```

**참고**: 
- API 키는 브라우저에 로컬로 저장되며 외부 서버로 전송되지 않습니다
- 두 API 키를 모두 입력하면 더 안정적인 분류가 가능합니다 (fallback 지원)

## 사용 방법

### 자동 추적 (설정 후 자동 작동)

설치와 설정이 완료되면 확장 프로그램이 자동으로 작동합니다:

1. **소셜 미디어 방문**
   - Instagram, Twitter/X, LinkedIn, TikTok 중 아무 곳이나 방문

2. **평소처럼 사용**
   - 게시물에 좋아요, 저장, 북마크, 리트윗 등 평소처럼 상호작용

3. **자동 저장**
   - 확장 프로그램이 상호작용을 자동으로 감지하고 저장

4. **AI 분류**
   - 저장된 콘텐츠는 백그라운드에서 자동으로 카테고리별로 분류됨 (몇 분 소요)

### 저장된 콘텐츠 보기

#### 빠른 보기 (팝업)

1. Chrome 툴바의 확장 프로그램 아이콘 클릭
2. 통계와 최근 저장 항목 확인
3. 항목 클릭하면 원본 게시물 열림

#### 대시보드 (전체 보기)

1. 팝업에서 "Open Dashboard" 클릭
2. 저장된 모든 콘텐츠 탐색
3. 플랫폼 또는 카테고리로 필터링
4. 검색 기능 사용
5. 날짜, 참여도, 시청 시간으로 정렬
6. 그리드/리스트 뷰 전환

### 콘텐츠 관리

- **원본 열기**: 카드 클릭하여 원본 게시물 열기
- **삭제**: 원하지 않는 저장 항목 제거
- **내보내기**: 대시보드에서 모든 데이터를 JSON으로 다운로드

## 주요 기능

### 1. 자동 콘텐츠 감지

- **MutationObserver**: DOM 변경을 감시하여 새 게시물 감지
- **이벤트 위임**: 좋아요/저장/북마크 버튼 클릭 감지
- **중복 방지**: 같은 게시물을 두 번 저장하지 않음

### 2. 시청 시간 추적

- **IntersectionObserver**: 화면에 보이는 게시물 추적
- **50% 가시성 기준**: 정확한 추적을 위한 임계값
- **탭 전환 시 일시정지**: 활성 탭에서만 시간 계산
- **누적 시간**: 여러 번 스크롤해도 시간 누적

### 3. AI 분류

- **큐 기반 처리**: API 속도 제한 준수
- **배치 처리**: 한 번에 5개 항목 처리
- **자동 재시도**: 실패 시 자동 재시도
- **50+ 카테고리**: Technology, Business, Health, Entertainment, Sports 등

### 4. 데이터 관리

- **로컬 저장**: Chrome Storage API 사용
- **무제한 용량**: unlimitedStorage 권한으로 제한 없음
- **내보내기/백업**: JSON 형식으로 데이터 다운로드
- **통계**: 사용 패턴 추적

## 문제 해결

### 확장 프로그램 아이콘이 보이지 않아요

**해결 방법 1: 확장 프로그램 고정**
- Chrome 툴바의 퍼즐 아이콘 클릭
- "Social Media Content Tracker" 찾기
- 핀 아이콘 클릭

**해결 방법 2: 활성화 확인**
- `chrome://extensions/` 방문
- 확장 프로그램 찾기
- 토글이 켜져 있는지 확인

### "확장 프로그램 로드됨" 메시지가 나오지만 작동하지 않아요

**해결 방법: 확장 프로그램 재로드**
1. `chrome://extensions/` 방문
2. "Social Media Content Tracker" 찾기
3. 원형 화살표(새로고침) 아이콘 클릭
4. 소셜 미디어 페이지 새로고침 (F5)
5. 다시 상호작용 시도

### API 키가 작동하지 않아요

**OpenAI의 경우:**
- 키 형식 확인: `sk-proj-...`
- 사용량 제한 확인: https://platform.openai.com/usage
- 결제 설정 확인

**Gemini의 경우:**
- Gemini Pro용 키인지 확인
- Google Cloud에서 API 활성화 확인
- https://makersuite.google.com/ 에서 테스트

### 콘텐츠가 저장되지 않아요

1. **브라우저 콘솔 열기** (F12)
2. **Console 탭 확인**
3. **오류 메시지 확인**
4. **일반적인 문제:**
   - "Failed to send message" → 확장 프로그램 재로드
   - "Storage quota exceeded" → 데이터 내보내기 후 오래된 데이터 삭제
   - "Selector not found" → 플랫폼 UI가 변경되었을 수 있음

### 카테고리가 나타나지 않아요

1. **5분 대기** (백그라운드 처리 시간)
2. **API 키 확인** (올바르게 입력되었는지)
3. **AI 제공자 확인** (키와 일치하는지)
4. **콘솔에서 API 오류 확인**
5. **수동 새로고침**: 팝업에서 "Refresh" 클릭

### 특정 플랫폼에서 작동하지 않아요

소셜 미디어 플랫폼은 UI를 자주 변경합니다:

1. **브라우저 콘솔 열기** (F12)
2. **선택자 오류 확인**
3. **확장 프로그램 재로드**
4. **페이지 새로고침**

## 팁과 권장사항

### 최상의 경험을 위한 팁

1. **설치 후 페이지 새로고침**: 트래커가 제대로 로드되도록
2. **즉시 API 키 설정**: 즉시 분류 기능 사용 가능
3. **정기적으로 내보내기**: 월 1회 데이터 백업 권장
4. **가끔 콘솔 확인**: 조기에 오류 발견
5. **플랫폼 업데이트 대비**: 플랫폼이 UI를 변경하면 선택자 업데이트 필요할 수 있음

### 데이터 관리

- **정기 백업**: 대시보드 → 내보내기로 JSON 파일 다운로드
- **저장 공간**: 무제한 저장이지만 오래된 데이터는 정리 권장
- **개인정보**: 모든 데이터는 브라우저에만 저장되며 외부 서버로 전송되지 않음

### 비용 관리

**OpenAI 사용 시:**
- 게시물당 약 $0.0001-0.001
- 100-500개 분류당 $1
- 대부분의 사용자에게 매우 저렴

**Gemini 사용 시:**
- 무료 티어: 분당 60개 요청
