# Quick Start Guide

Get your Social Media Content Tracker up and running in 5 minutes!

## Step 1: Load Extension in Chrome

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right)
3. Click "Load unpacked"
4. Select this folder: `/Users/zone/Chrome_extension`
5. Extension should appear with a purple bookmark icon

## Step 2: Get an API Key

Choose ONE of these options:

### Option A: OpenAI (Recommended - More Accurate)

1. Visit: https://platform.openai.com/api-keys
2. Sign up/login
3. Click "Create new secret key"
4. Copy the key (starts with `sk-...`)

**Cost**: ~$0.0001-0.001 per post (very cheap!)

### Option B: Google Gemini (Free Tier Available)

1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with Google
3. Click "Create API key"
4. Copy the key

**Cost**: Free for 60 requests/minute

## Step 3: Configure Extension

1. Click the extension icon in Chrome toolbar (purple bookmark)
2. Click "Settings"
3. Fill in:
   - **AI Provider**: Choose "OpenAI" or "Google Gemini"
   - **API Key**: Paste your key
   - **Time Threshold**: Leave at 5 seconds (default)
   - **Enabled Platforms**: Keep all checked
4. Click "Save Settings"

## Step 4: Test It Out!

1. Go to Instagram, Twitter, LinkedIn, or TikTok
2. Like or save a post
3. Click the extension icon
4. You should see the post in "Recent Saves"!

## That's It!

The extension is now tracking your interactions automatically.

### What Happens Next?

- Every like/save/bookmark is captured
- AI categorizes content in the background (takes a few minutes)
- View all saved content in the Dashboard
- Search, filter, and export your content anytime

## Need Help?

### Extension not working?

1. Refresh the social media page after installing
2. Check browser console (F12) for errors
3. Verify API key is correct in settings

### No categorization?

1. Wait 5 minutes (AI processes in background)
2. Check API key and provider match
3. Verify you have API credits remaining
